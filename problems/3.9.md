## 题目名称
超大整数幂计算与数字统计
### 题目描述
计算 2 的 4000 次方，并统计结果中数字 0~9 各自出现的次数。  
要求使用链表存储高精度整数，运算逻辑仿照多项式运算实现。  
- 输入格式  
无输入  
（固定计算 2^4000）  
- 输出格式  
依次输出 0~9 每个数字的出现次数，每行一个：   
0: x1  
1: x2  
...  
9: x10  

---

### 解法一：循环4000次
```cpp
#include<iostream>
using namespace std;

struct Node
{
    int val;
    Node* Next;
    Node(int v, Node* ptr = nullptr): val(v), Next(ptr) {}
};

typedef Node* PtrToNode;
typedef PtrToNode BigInt;
typedef PtrToNode Position;

int cnt[10]{0};

void MultiplyByTwo(BigInt & bigint) //只有一位的int 乘以 大整数
{
    int carry = 0;
    Position p, Pre;

    for (p = bigint; p; p = p->Next)
    {
        int curVal = p->val * 2 + carry;
        carry = curVal / 10;
        p->val = curVal % 10;
        Pre = p;
    }
    if (carry > 0)
        Pre->Next = new Node(carry);
}

int main(void)
{
    Position Pos;
    int i;
    BigInt bigint = new Node(1);
    Pos = bigint;
    for (i = 0; i < 4000; ++i)
        MultiplyByTwo(bigint);
    while (Pos)
    {
        Position tmp = Pos;
        cnt[Pos->val]++;
        Pos = Pos->Next;
        delete tmp;
    }
    for (i = 0; i < 10; ++i)
        cout << i << ":" << cnt[i] << endl;
}
```
