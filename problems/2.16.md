## 快速求幂
> $O(logN)$ 
> 在 $O(1)$ 的时间内将范围缩小了一半


- 递归实现
```c
long long 
Pow(long long X, unsigned int N)
{
	/* base case */
	if ( N == 0 )
		return 1;
	if ( N % 2 == 1 )
		return X * Pow(X, N - 1);
	else
		return Pow(X * X, N / 2);
}
```
- 非递归实现

```c
long long
Pow(long long X, unsigned int N)
{
    long long result = 1;

    while (N > 0)
    {
        if (N % 2 == 1)
            result *= X;
        X *= X;
        N /= 2;
    }
    return result;
}
```
