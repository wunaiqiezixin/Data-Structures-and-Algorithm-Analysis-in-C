# 给出运行时间分析(使用大O)
### (1)
```c
Sum = 0;
for (i = 0; i < N; i++)
	Sum++;
```
#### 解：
$$O(N)$$

### (2)
```c
Sum = 0;
for (i = 0; i < N; i++)
	for (j = 0; j < N; j++)
		Sum++;
```
#### 解：
主要关注第四行的语句`Sum++` 执行次数
$$
\sum_{i = 0}^{N - 1} \sum_{j = 0}^{N - 1} 1 
= \sum_{i = 0}^{N - 1} N = \dfrac{(0 + N - 1) (N)}{2}
$$

$$
O(\frac{N(N - 1)}{2})
$$
$$
O(N^2)
$$
### (3)
```c
Sum = 0;
for (i = 0; i < N; i++)
	for (j = 0; j < N * N; j++)
		Sum++;
```

#### 解：
关注 `Sum++` 的执行次数
$$ 
\sum_{i = 0}^{N - 1} \sum_{j = 0}^{N^2 - 1} 1
 = \sum_{i = 0}^{N - 1} N^2 
 = N(N^2)
 = N^3
$$
$$
O(N^3)
$$


### (4)
```c
Sum = 0;
for (i = 0; i < N; i++)
	for (j = 0; j < i; j++)
		Sum++;
```

#### 解：

$$
\sum_{i = 0}^{N - 1} \sum_{j = 0}^{i - 1} 1
= \sum_{i = 0}^{N - 1} i
= \dfrac{(0 + N - 1)N}{2} 
= \dfrac{N(N - 1)}{2}
$$
$$
O(\dfrac{N(N - 1)}{2})
$$
$$
O(N^2)
$$
### (5)
```c
Sum = 0;
for (i = 0; i < N; i++)
	for (j = 0; j < i * i; j++)
		for (k = 0; k < j; k++)
			Sum++;
```

#### 解：
$$
T(N) = 
\sum_{i = 0}^{N - 1} \sum_{j = 0}^{i^2 - 1} \sum_{k = 0}^{j - 1} 1
= \sum_{i = 1}^{N - 1} \sum_{j = 0}^{i^2 - 1} j 
= \sum_{i = 1}^{N - 1} \dfrac{(0 + i^2 - 1)(i^2 - 1)}{2} 
= \frac{1}{2} \sum_{i = 0}^{N - 1} (i^2 - 1)^2
$$
$$
T(N) = \frac{1}{2} \sum_{i = 0}^{N - 1} i^4 - \sum_{i = 0}^{N - 1} i^2 + 
\frac{1}{2} \sum_{i = 0}^{N - 1} 1
$$
$$
T(N) = 
\sum_{i = 0}^{N - 1} i^4 
= \dfrac{(N - 1)N(2N - 1)(3N^2 - 3N - 1)}{30} \approx 
\frac{2}{15} N^5
$$
$$
T(N) = O(N^5)
$$

### (6)
```c
Sum = 0;
for (i = 1; i < N; i++)
	for (j = 1; j < i * i; j++)
		if (j % i == 0)
			for (k =  0; k < j; k++)
				Sum++;
```

#### 解：

##### 最内层循环, 如果满足 `j % i == 0` 会执行 `k` 次 
##### 满足条件的 `j` 为 `i, 2i, 3i.....(i - 1)i` , 共 `i - 1` 个
##### 对于满足条件的每一个 `i` `(j = mi)` , `k` 循环 (`Sum++`) 的执行次数为

$$
\sum_{m = 1}^{i - 1} mi = i\frac{(1 + i - 1)(i - 1)}{2} 
= \frac{i^2(i - 1)}{2}
$$
##### 外层 `i` 循环，计算 `Sum++` 总的执行次数 
$$
\sum_{i = 1}^{N - 1} \frac{i^2(i - 1)}{2}
= \frac{1}{2}\sum_{i = 1}^{N - 1} i^3 - \frac{1}{2} \sum_{i = 1}^{N - 1} i^2
$$
##### 所以 `Sum++` 总的执行次数为
$$
\dfrac{N(N - 1)(3N^2 - 7N + 2)}{24}  \approx \frac{N^4}{8}
$$

##### 再来看看 `if (j % i == 0)` 的执行次数
$$
\sum_{i = 1}^{N - 1} \sum_{j = 1}^{i^2 - 1} 1 
= \sum_{i = 1}^{N - 1} (i^2 - 1) 
= \dfrac{N(N - 1)(2N - 1)}{6} - N + 1 
\approx \frac{N^3}{3}
$$
##### 时间复杂度为
$$
T(N) = O(\frac{N^4}{8}) + O(\frac{N^3}{3})
$$
$$
T(N) = O(N^4)
$$
