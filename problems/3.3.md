### doubly-linked list
```cpp
void SwapAdjacentNode(Position First, Position Last)
{
    First->next = Last->next;
    if (Last->next)
        Last->next->pre = First;

    Last->next = First;
    Last->pre = First->pre;

    if (First->pre)
        First->pre->next = Last;
    First->pre = Last;
}
```
