### 实现两个多项式相乘的程序
```cpp
/**
 * Definition for singly-linked list.
 * typedef Node* PtrToNode;
 * typedef PtrToNode List;
 * typedef PtrToNode Poly;
 * struct Node
 * {
 *    int Coefficient;
 *    int Exponent;
 *    PtrToNode Next;
 *    Node(): Coefficient(-1), Exponent(-1), Next(nullptr) {}
 *    Node(PtrToNode ptr): Coefficient(ptr->Coefficent), 
 *                          Exponent(ptr->Exponent), Next(nullptr) {}
 *    Node(int coeff, int expo): Coefficient(coeff), Exponent(expo), 
 *                                  Next(nullptr) {}
 * };
 */
```cpp
//返回两个多项式P1、P2相乘的结果
Poly Multiply(Poly P1, Poly P2);
```

### 一般想法：多项式自乘P次
```cpp
Poly PolyPower(Poly F, unsigned int P)
{
    Poly newPoly = new Node(1, 0);
    int i;
    for (i = 0; i < P; ++i)
    {
        Poly tmp = Multiply(F, newPoly);
        newPoly = tmp;
    }
    return newPoly;
}
```

### 多项式的快速幂
>递归法
```cpp
Poly PolyPower(Poly F, unsigned int P)
{
    if (P == 0)
    {
        Poly tmp = new Node(1, 0);
        return tmp;
    }
    if (P % 2 == 1)
        return Multiply(F, PolyPower(F, P - 1));
    else
        return PolyPower(Multiply(F, F), P / 2);
}
```

>迭代法
```cpp
Poly PolyPower(Poly F, unsigned int P)
{
    Poly result = new Node(1, 0);
    while (P > 0)
    {
        if (P % 2 == 1)
            result = Multiply(result, F);
        F = Multiply(F, F);
        P /= 2;
    }
    return result;
}
